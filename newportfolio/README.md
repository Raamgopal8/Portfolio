# Raamgopal S | Software Developer Portfolio

A premium, modern portfolio showcasing the intersection of clean architecture, intelligent systems, and architectural precision. Built with Next.js 16 and Tailwind CSS 4, this project features a dual-experience interface: a minimalist project showcase and an AI-powered assistant.

## 🚀 The Experience

The portfolio is architected as a gateway to two distinct digital dimensions:

- **Digital Exploration (Archive)**: A curated gallery of selected works in AI, Cloud, and Scalable Systems, presented through a timeless, minimalist lens.
- **AI Assistant (Curator)**: An intelligent chatbot powered by state-of-the-art LLMs, designed to provide dynamic insights and interactive walkthroughs of my technical journey.

## 🛠️ Getting Started

### Prerequisites

- Node.js 18.x or later
- npm (v9+) or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Raamgopal8/Portfolio.git
   cd Portfolio/newportfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) to explore.
